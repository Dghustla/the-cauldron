# 🫕 The Cauldron — Cowork Deployment Guide

Complete step-by-step instructions to get The Cauldron live on Netlify from your local machine.

---

## Prerequisites

Before starting, make sure you have:
- [ ] **Node.js 18+** — check with `node -v`
- [ ] **Git** — check with `git --version`
- [ ] **A GitHub account**
- [ ] **A Netlify account** (free tier is fine — https://netlify.com)
- [ ] **Your Anthropic API key** — from https://console.anthropic.com
